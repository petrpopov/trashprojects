Examples

Target.Health - target's current health level.

Target.x - target's x coordinate in the robot relative coordinate system.

Target.y - target's y coordinate in the robot relative coordinate system.

Target.Distance - distance toward the target (how many moves you need to reach it)

Target.Probability - probability of hitting the target by shot. (Equals 100% if the target can be seen directly and rapidly decreases with distance when shooting by hearing.)
