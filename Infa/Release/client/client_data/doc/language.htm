Robot logic(program) is a set of conditions and commands. On each turn robot analyses the situation and makes some actions according to it's program, until turn time runs out or robot reaches the end of his program.

Pay attention to the fact that the situation analysis goes each turn anew! And so, each turn the program runs from the very beginning!

Reactions of a robot for the changing situation can be assigned by means of condition blocks:

if ( condition1 ) {
actions 1
}elseif ( condition2 ){
actions 2
...
}elseif( conditionN ) {
actions N
}else {
default actions
}

Each such construction must start with an "if" keyword. Conditions are enclosed in round brackets, action blocks - in braces. Such construction may (unnecessarily) contain any number of alternative choice blocks "elseif () {}" and an "else {}" block. Only one block of this structure will be executed - the first one to have the right condition (or the "else {}" block if no conditions are right).

You can also use nested condition blocks.

The program should end with a "stop" keyword. You may also use the "stop" keyword if you need to stop the execution of the program in some situation.

You'll find more detailed information about all the language functions in the corresponding help sections.

Robot language is case-insensitive.
