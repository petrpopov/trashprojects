This two functions allow to easily reach the finish, but only if a robot can see it directly! Use them only together.

FinishIsKnown - returns true, if the finish was spotted and false in the opposite case.

GoToFinish - a command to make a step towards the finish. Do not use this function separately from FinishIsKnown!

Example:

if ( FinishIsKnown ) { GoToFinish }
