Attention! This properties may be used only for assigning priorities to the waypoints!

Waypoint.Distance - distance toward the current waypoint.

Waypoint.MinNewDist - distance from current waypoint to another one, closest to it.

Waypoint.MaxNewDist - distance from current waypoint to another one, farthest from it.

Waypoint.AvgNewDist - average distance from current waypoint to another ones.
