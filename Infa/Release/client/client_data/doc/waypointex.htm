What is a waypoint? In this case waypoint means an entrance into an unexplored corridor. In other words it is such a known cell within the robot's reach which has at least one unexplored cell next to it.

WaypointExists (line1, line2) - this function makes the robot examine a sector between two directions given as parameters. Directions are numbers of the rays in the relative polar coordinate system . If a robot finds any waypoints visible in the sector, the function returns true, in other cases - false.
