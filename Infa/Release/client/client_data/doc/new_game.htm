Click the "Create New Game" button. A settings window will appear.

Game type - choose the gamemode: a search in a labyrinth or a battle. For details about gamemodes see game types .

Game name - the name of the game :). When other players join the game you created, they'll see this name.

Number of players - The game will start only when the necessary number of players joins it. You may specify it as 1 and play just for yourself.

Game logik - the path to a file with your robot logic. About writing the logic see here. The robot language is described in the Language overview

Server IP - IP-address of the server where the game will run. If the server is on your computer you need to input 127.0.0.1. If you want to add new server to the list, select "Add server" from the menu, input the new server IP and click "Add".

Server port - the port server uses. Default value is 51000. Don't change this value if you're unsure what you're doing!

You can not choose the map for the game. It will be chosen randomly by the server from all the maps it has. We think that's more honest. The point is that the players won't write the logic designed especially for a certain map and a certain situation. If you don't have the necessary map on your computer it will be automatically loaded from the server.

After you've brought in all the necessary data, the "Create game" button becomes available. On pressing this button you connect to the server and create a game on it. If everything goes right you'll see a window saying "Waiting". When the necessary number of players joins the game you'll see a picture of your robot and everything it sees and it knows. In this window you may watch the game, the actions of your robot and the other ones. In the and of the game there will be a message about the winner.
