Installing and setting the game server is in fact very simple! :) It won't take longer than 10 minutes.

Step1. Copy the content of the "server" directory into the directory, where you plan to place the server and all it's data.

Step2. Compile the server. To do this run the console and call the directory where you have the server. Then type a command "gcc server.c -o server.out"

Step3. You may delete the files server.c and parser.h which are not necessary any more.

Step4. Server settings. All the server settings are in the gameserv.conf file. For changing the settings just edit the file (all instructions included). In most cases the default settings fit a small "for friends" server/

Step5. That's all ! You may now play!. For starting the server just run the file server.out you created on step2.
