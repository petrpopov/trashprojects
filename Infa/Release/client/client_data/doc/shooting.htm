These functions may be used for shooting:

SetEnemiesPriority ( equation ) - the function sets priorities to all known waypoints according to an equation it has been given as an argument. In this equation you may use any properties , integer numbers, operators +,-,*,/, and also brackets.

Example: SetEnemiesPriority ( Enemies.HealthMax - Enemy.Health ). Each enemy this way will get as a priority the difference between the health of the strongest robot and the the health of the current one. So the weakest enemy will receive the highest priority.

GetTarget - this function chooses an enemy with the highest priority and sets it as a target.

Shoot - Robot shoots the current target, it's turn ends.
