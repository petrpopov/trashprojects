Language overview

 

For playing you need the game logic in a text file. In fact you may write it in any text editor, but what for?

We propose that you use the built-in code editor. With it's help you may by 2 clicks insert any functions of the robot language, which guarantees no mistypes. Besides you get access to the context help on any function. After you have created the logic, you may check it by clicking the "Check logik" button. If you've made any mistakes, you'll get a message about the mistake type and the number of the wrong string. After your logic is ready you may save it and use in the game.
