At first you need to find a game to join. Click the "Find Game" button. A window with settings appears.

Game type - choose the gamemode: a search in a labyrinth or a battle. For details about gamemodes see game types .

Server IP - IP-address of the server where the game will run. If the server is on your computer you need to input 127.0.0.1. If you want to add new server to the list, select "Add server" from the menu, input the new server IP and click "Add".

Server port - the port server uses. Default value is 51000. Don't change this value if you're unsure what you're doing!

After you press the "Find game" button in a window to the left you'll see the list of available games. There will be shown the Name of the game, full number of players in the Number field and current number of players in the Current field.

For joining the game choose one from the list. Then fill the Game logik field with a path to the file with your robot logic. About writing the logic see here. The robot language is described in the Language overview

After you've brought in all the necessary data, the "Join game" button becomes available. On pressing this button you connect to the server and join the game you've chosen. If everything goes right you'll see a window saying "Waiting". When the necessary number of players joins the game you'll see a picture of your robot and everything it sees and it knows. In this window you may watch the game, the actions of your robot and the other ones. In the and of the game there will be a message about the winner.
