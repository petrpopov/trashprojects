Behaviour of all this functions depends on the drivemode a robot currently has. If a robot's drivemode is normal, then only wall cells will count as inaccessible. If the current drivemode is careful, then all the cells where a collision with another robot is possible will count as inaccessible too. The drivemodes are set using the SetDriveMode ( mode ) function.

CanGoForward - returns true if a cell ahead is accessible and false in the opposite case.

CanGoLeft - returns true if a cell to the left is accessible and false in the opposite case.

CanGoRight - returns true if a cell to the right is accessible and false in the opposite case.

CanGoBack - returns true if a cell backwards is accessible and false in the opposite case.

IsAvailablePoint (x, y) - returns whether a cell with relative cartesian coordinates x,y (x and y are integer numbers) is accessible or not.
